# o2o优惠券使用预测

## 商户相关特征
### GetMerchantRelatedFeature
* 商户卖出商品（有户有或无优惠券）的数量
* 商户卖出商品（用户使用优惠券）的数量
* 商户优惠券被领取总数量
* 商户与使用优惠券消费的用户的距离（最小，最大，平均，中位数）
* 商户卖出商品中优惠券的占比
  * 商户卖出商品（用户使用优惠券）的数量 / 商户卖出商品（有户有或无优惠券）的数量
* 商户卖出商品（用户使用优惠券）占领取优惠券总比例
  * 商户卖出商品（用户使用优惠券）的数量 / 商户优惠券被领取总数量

## 其他特征（预测区间内提取）
### GetOtherFeature
* 用户领取优惠券的总数量
* 用户领特定优惠券的领取数量  
。。。
